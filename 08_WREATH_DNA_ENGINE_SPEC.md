# 08_WREATH_DNA_ENGINE_SPEC

## Purpose

The Wreath DNA Engine turns emotional and formula data into procedural design traits used by the Placement Engine.

## Output Example

```json
{
  "cluster_count": 3,
  "cluster_weight_distribution": 0.62,
  "cluster_spread_deg": 70,
  "density_profile": 0.58,
  "greenery_direction": "outward",
  "silhouette_bias": "organic",
  "greenery_ratio": 0.44,
  "focal_ratio": 0.22,
  "silence_arc": 48,
  "focal_depth": 0.72,
  "style_signature": "editorial",
  "color_bias": "cool_warm_split"
}
```

## MVP DNA Traits

cluster_count, density_profile, cluster_spread_deg, greenery_direction, focal_ratio, silence_arc, silhouette_bias, style_signature.

## Role in the System

Emotion tells the system what it should feel like. Formula tells the system what structural archetype to use. DNA tells the Placement Engine how that archetype should behave.
