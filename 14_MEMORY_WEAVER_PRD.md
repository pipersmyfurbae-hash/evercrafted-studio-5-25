# 14_MEMORY_WEAVER_PRD

## Purpose

Memory Weaver is the emotional intake experience. It transforms written memories, moods, and design requests into structured design direction.

## User Flow

1. User enters memory/mood/concept.
2. System classifies emotion.
3. System recommends formula.
4. System generates design direction.
5. System sends data to Blueprint Compiler.

## MVP Inputs

text prompt, season, wreath size, optional palette preference, optional style preference.

## MVP Outputs

emotional summary, palette direction, recommended formula, density, material suggestions, blueprint generation button.

## Key UX Requirement

Memory Weaver should use plain language and not expose engine complexity unless the user chooses advanced view.
