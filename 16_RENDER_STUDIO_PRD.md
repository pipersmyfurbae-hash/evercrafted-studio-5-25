# 16_RENDER_STUDIO_PRD

## Purpose

Render Studio converts blueprint data into visual prompt outputs and product photography directions.

## MVP Scope

Generate AI image prompt, product photo direction, white-background product prompt, lifestyle scene prompt, and Etsy-ready alt text.

## Inputs

EC_CANON_V1 blueprint, style mode, aspect ratio, scene type.

## Rule

Render Studio must preserve faux floral realism and avoid fresh bouquet language.
