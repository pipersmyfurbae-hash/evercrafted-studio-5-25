# 17_DESIGN_LIBRARY_PRD

## Purpose

Design Library stores blueprints, versions, tags, and exports.

## MVP Features

- list saved blueprints
- open blueprint
- save blueprint
- duplicate blueprint
- export JSON
- search by name, mood, season, formula

## Data Stored

EC_CANON_V1 object, user ID, created timestamp, version, tags, and thumbnail/preview if available.
