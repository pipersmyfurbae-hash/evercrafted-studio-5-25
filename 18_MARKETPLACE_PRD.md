# 18_MARKETPLACE_PRD

## Purpose

The Marketplace allows creators to publish and sell blueprint-based design products.

## V2 Scope

Not MVP.

Marketplace should eventually support free blueprint downloads, premium blueprint sales, construction guide add-ons, seller revenue share, mood/season/style filters, and preview thumbnails.

## MVP Note

MVP should only prepare data so saved blueprints can become marketplace-ready later.
