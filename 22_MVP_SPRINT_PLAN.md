# 22_MVP_SPRINT_PLAN

## Purpose

This is the build-order document for developers.

## Sprint 1 — Foundation

Repo setup, frontend shell, backend shell, database setup, auth, environment variables.

## Sprint 2 — Canon + Blueprint Storage

EC_CANON_V1 types, blueprint table, blueprint save/load, JSON validation.

## Sprint 3 — Emotion + Formula Pipeline

Emotion classify endpoint, formula recommendation, Memory Weaver MVP UI.

## Sprint 4 — Placement Engine

Deterministic placement generator, polar coordinate output, placement validation, blueprint compile endpoint.

## Sprint 5 — Visualization Engine

Canvas/SVG preview, radial grid, layer rendering, placement inspector.

## Sprint 6 — Design Studio MVP

Open blueprint, preview blueprint, inspect placements, regenerate placement, save/export.

## Sprint 7 — Inventory + BuildBloom

Inventory CRUD, material assignment, build sheet generator, printable build output.

## Sprint 8 — QA + Launch Prep

Test flows, bug fixes, deployment, documentation, demo data.

## MVP Demo Flow

Memory → Emotion → Formula → DNA → Placement → Visualization → Blueprint → Build Sheet.
