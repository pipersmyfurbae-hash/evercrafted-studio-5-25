# 23_TECH_STACK_AND_DEPLOYMENT

## Recommended Stack

### Frontend
Next.js, React, TypeScript, Tailwind CSS or tokenized CSS, Canvas/SVG visualization.

### Backend
FastAPI or Node/TypeScript, PostgreSQL, JSONB blueprint storage, server-side AI calls.

### Hosting
Vercel frontend, Render/Railway backend, Supabase/Neon database, Cloudflare R2 for file storage later.

## Environment Variables

- DATABASE_URL
- AI_API_KEY
- JWT_SECRET
- NEXT_PUBLIC_API_URL

## Deployment Rule

Never expose AI API keys to the client.
