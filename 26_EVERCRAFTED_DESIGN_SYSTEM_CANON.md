# 26_EVERCRAFTED_DESIGN_SYSTEM_CANON

## Purpose

This file defines the Evercrafted visual system. The uploaded System Console establishes the internal OS visual language.

## Visual Identity

Evercrafted OS uses dark luxury interface, gold accents, muted botanical tones, serif display type, monospaced system text, modular panels, and diagnostic layouts.

## Design Feeling

The interface should feel like creative intelligence software, luxury infrastructure, botanical operating system, and premium design console.

It should not feel like hobby craft software, generic SaaS dashboard, bright social media app, or childish seasonal template tool.

## Component Doctrine

Use cards, panels, modules, status badges, radial visuals, JSON inspectors, engine maps, and console indicators.

## Studio vs OS

Evercrafted OS can be more technical and dark. Evercrafted Studio may become slightly softer, clearer, and creator-friendly while preserving premium brand DNA.
