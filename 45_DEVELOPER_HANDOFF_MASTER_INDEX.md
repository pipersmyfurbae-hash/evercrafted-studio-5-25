## Do Not Build Yet
Do not build in MVP:
- full marketplace
- Moodoor
- adaptive intelligence
- psychographic clustering
- Shopify publishing
- Gmail/Calendar integrations
- advanced analytics
- team collaboration

## MVP Definition
The MVP is complete when a user can:
1. Enter memory/mood text.
2. Generate structured emotion output.
3. Compile a canonical blueprint.
4. Generate deterministic placement data.
5. View placement in Design Studio.
6. Save to Design Library.
7. Generate BuildBloom build sheet.
8. Generate Render Studio prompt.
