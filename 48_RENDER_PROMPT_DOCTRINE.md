# 48_RENDER_PROMPT_DOCTRINE

## Purpose

This file defines the official rendering prompt doctrine for Evercrafted.

---

## Core Rule

Evercrafted render prompts must preserve:

- premium faux floral realism
- believable stem direction
- asymmetrical composition
- editorial styling
- visible movement
- realistic spacing
- manufacturable structure

---

## Forbidden Prompt Characteristics

Avoid:
- fresh florist bouquet styling
- centered flower blobs
- excessive symmetry
- unrealistic density
- “AI art” aesthetics
- watercolor/cartoon language
- giant floral explosions

---

## Required Prompt Structure

Every render prompt should contain:

1. Base description
2. Formula structure
3. Placement direction
4. Floral palette
5. Greenery movement
6. Density level
7. Faux floral realism language
8. Photography direction
9. Negative prompts

---

## Preferred Language

Use:
- premium faux florals
- silk floral realism
- editorial botanical styling
- airy movement
- asymmetrical crescent
- visible grapevine structure
- layered greenery
- restrained luxury

Avoid:
- bouquet
- fresh-cut flowers
- wedding arrangement
- perfectly balanced
- centered composition
