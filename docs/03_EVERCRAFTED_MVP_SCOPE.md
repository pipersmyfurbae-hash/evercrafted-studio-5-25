# 03_EVERCRAFTED_MVP_SCOPE

## MVP Goal

Build the smallest working version of Evercrafted Studio that proves the core loop:

Memory → Blueprint → Placement → Visualizer → Save → Build Sheet

## MVP Includes

1. App foundation
2. Authentication foundation
3. Blueprint storage
4. Memory Weaver
5. Deterministic Placement Engine
6. Design Studio visualizer
7. Design Library
8. Basic Inventory
9. BuildBloom build sheet

## MVP Does Not Include

- Marketplace
- Moodoor
- Adaptive Intelligence
- Enterprise features
- Shopify/Etsy publishing
- Full Admin OS
- Advanced analytics

## Core Doctrine

AI interprets. Geometry places.

AI may classify emotion and explain design intent.

Geometry/code must handle placement, coordinates, spacing, and structure.

## MVP Success Test

A user can:

1. Enter a memory or mood.
2. Generate a structured blueprint.
3. Generate deterministic placement data.
4. See the layout visually.
5. Save the blueprint.
6. Generate a build sheet.
